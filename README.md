# IHM-project
projet d'IHM de L3 Informatiques

## Compilation ##
En ligne de commande 
* sous linux `./compil.sh`
* sous Windows `.\compil.bat`

## Execution ##
En ligne de commande 
* sous linux `./run.sh`
* sous Windows `.\run.bat`

### logs et erreurs ###
* sortie standard : **log.log** 
* sortie d'erreur : **err.log**
